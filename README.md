## Dataset
- Source: MBIE Tenancy Services (tenancy.govt.nz)
- Licence: Creative Commons Attribution 3.0 New Zealand
- Coverage: 67 locations × 398 months (Feb 1993 – Mar 2026)
- Variables: Median Rent, Geometric Mean, IQR, Lodged/Active/Closed Bonds
